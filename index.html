VideoFocus-R1: Agentic RL with Dynamic Spatio-Temporal Focus for Long Video Understanding

Abstract

Despite the advent of reasoning multimodal large language model (MLLM), chain-of-thought for long video understanding remains challenging due to long context and complex spatio-temporal information. Existing reinforcement learning (RL) methods for video reasoning MLLM rely on fixed uniform frame sampling, limiting their ability to focus on key events or prominent objects. Such a limitation prohibits their direct use in long video settings, where fixed-frame samples provide only sparse and mostly irrelevant information. In this work, we present VideoFocus-R1, a reinforced agentic model for effective long video reasoning with focused spatio-temporal evidence. We propose an agent-style reasoning process that employs dynamic video processing tools, including a tool to search for key events for slow-motion playback and another tool for detecting prominent objects to highlight them. To facilitate effective supervised fine-tuning (SFT) training for cold-start, we create VideoFocus-SFT-10k, a high-quality video tool-use dataset developed using a carefully designed automated synthesis pipeline. To improve generalization and build a robust agent model, we leverage a Group Relative Policy Optimization (GRPO)–based algorithm for video agents to enhance the model’s reasoning capability. Through extensive quantitative evaluations, we demonstrate that VideoFocus-R1 significantly improves performance on both regular-length and long-video question-answering benchmarks. Moreover, we showcase that the proposed model achieves a reasoning process with explicit spatio-temporal evidence.

Overview

The policy model is first initialized with an SFT-based cold start and subsequently optimized through GRPO-based agent reinforcement learning. During RL, the video, question, and an initial prompt defining the available tools are fed into the policy model, which generates G groups of rollout trajectories. In each trajectory, the policy model generates step-by-step thoughts and tool calls, producing the final answer at the last step. These tool calls are parsed, executed online, and their results are incorporated into the next reasoning step. Finally, the policy model is updated using rule-based rewards.

Dynamic video processing is realized through temporal and spatial visual tools during the rollout process. For key events identified during the model's reasoning, the model can invoke event search tools to get key time ranges and perform slow-motion, high-FPS playback. For key objects identified during reasoning, the model can invoke object detection tools to track them with bounding boxes and textual labels.

Qualitative results

Here we show some examples of agents in action. During reasoning, VideoFocus-R1 uses the segment search tool to play key events in slow motion and utilizes the object detection tool to highlight critical objects. This facilitates accurate final answers while providing explicit spatiotemporal visual evidence.
